# 图书馆座位预约小程序

微信小程序项目，实现图书馆座位预约、暂离、黑名单、历史记录与续约功能。

## 功能说明

| 功能 | 说明 |
|------|------|
| **座位预约** | 选择楼层（A/B/C）、座位号，选择预约时长（2/4/6/8小时），黑名单用户不可预约 |
| **申请暂离** | 对当前在用的预约申请暂离（15/30/45/60 分钟），暂离期间可“返回座位”恢复使用 |
| **黑名单管理** | 按用户ID添加/移除黑名单，黑名单用户无法预约座位 |
| **历史记录查询** | 查看当前用户的预约历史（进行中、已结束、已取消） |
| **座位续约** | 对当前预约延长 1/2/3/4 小时，也可取消当前预约 |

## 运行方式

1. 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，选择「导入项目」
3. 目录选择本项目的 `library-seat-miniprogram` 文件夹
4. AppID 可先使用测试号或留空（体验版）
5. 编译并预览/真机调试

## 数据说明

- 数据保存在本地 `wx.setStorageSync`，仅作演示与课程作业使用。
- 首次启动会初始化 A/B/C 三楼，每楼 20 个座位（A01–A20, B01–B20, C01–C20）。
- 用户以本地生成的 `userId` 区分，未做真实登录；黑名单、历史记录均按该 ID 判断。

## 注意事项

- 暂离超时或预约结束时间到达后，再次进入相关页面时会自动释放座位（在 `releaseExpiredReservations` 中处理）。
- 黑名单管理为管理端功能，实际项目中应做权限控制并与后台同步。
